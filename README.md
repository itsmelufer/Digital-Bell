# School Bell

![py](https://img.shields.io/badge/Python-3.13+-blue)
![pygame](https://img.shields.io/badge/Pygame-2.6.1-green)
![pyinstaller](https://img.shields.io/badge/PyInstaller-6.15.0-orange)

Users can select their own audio file for the bell and create a schedule for automatic bell ringing.

---

## Features

- User-friendly interface
- Automatic bell scheduling
- Manual bell trigger
- Activity logging
- Auto-save settings

---

## How to Build and Run

1. **Clone the repository**:

   ```bash
   git clone <your-repo-url>
   cd SchoolBellSystem

### Double-click build.bat or run it in Command Prompt:
   ```bash
   build.bat
```
